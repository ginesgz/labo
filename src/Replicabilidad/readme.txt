Para replicar los resultados obtenidos en Kaggle, ejecutar los scripts en orden.

Script 00: actualiza los valores en pesos por IPC, y agrega una nueva variable para convertir a dolares las variables que estaban pesificadas.

Script 01: realiza feature engineering para enriquecer el dataset.

Script 02: realiza optimización bayesiana para obtener hiperparámetros del modelo final.

Script 03: genera el modelo final y los archivos con predicción por corte para subida a Kaggle .
